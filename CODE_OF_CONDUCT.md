# Code of Conduct

## Our Pledge

In the interest of fostering an open and welcoming environment, contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- using respectful, constructive language,
- accepting helpful feedback gracefully,
- focusing on technical issues rather than personal attacks,
- showing empathy toward other contributors.

Examples of unacceptable behavior:

- abusive or harassing language,
- personal attacks,
- trolling or deliberately inflammatory discussion,
- publishing private information without permission.

## Enforcement

Project maintainers are responsible for clarifying standards and may remove or reject contributions that do not align with this Code of Conduct.

## Scope

This Code applies within project spaces and in public spaces when an individual is representing the project.
